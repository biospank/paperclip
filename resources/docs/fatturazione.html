Permette l'inserimento, modifica e cancellazione dei proforma.

Inserimento

Premere il pulsante Nuovo F12, inserire i dati del cliente e premere il pulsante Salva F8. Seguire le indicazioni in caso di dati mancanti o incompleti.

Fattura

Permette l'inserimento, modifica e cancellazione delle fatture di vendita.

Inserimento

La fattura si compone dei seguenti dati obbligatori:

  1. Cliente
  2. Numero fattura (generata dal sistema)
  3. Data fattura (generata dal sistema)
  4. Righe fattura

Inserire i dati del cliente premendo il pulsante Cliente F6.
Verificare il numero e la data della fattura.

Le righe della fattura si compongono dei seguenti dati obbligatori:

  1. Descrizione del prodotto o servizio
  2. Importo da incassare al netto dell'aliquota iva
  3. Aliquota iva
Posizionare il cursore sulla descrizione della riga della fattra. Inserire la descrizione, l'importo e l'aliquota (vedi gestione aliquote). Premere aggiungi o invio per confermare i dati. Per modificare una riga, selezionarla dalla lista e premere modifica o invio dopo aver modificato i dati. Spuntare la voce Nota di credito se si tratta di una nota di credito.
Spuntare la voce ritenuta per applicare la ritenuta al conteggio della fattura. Nel campo codice ritenuta premere F5 per la ricerca della ritenuta da applicare (vedi gestione ritenute).
Inserire la descrizione Seguire le indicazioni in caso di dati mancanti o incompleti. Premere il pulsante Nuovo F12 per ripulire la maschera.

Modifica

Nota: La modifica della fattura è possibile solo nel caso in cui non sia già stata registrata in prima nota.

Premere il pulsante Variazione F3, ricercare la fattura nella maschera di ricerca fatture clienti. Modificare i dati e premere il pulsante Salva F8. Seguire le indicazioni in caso di dati mancanti o incompleti.

Cancellazione

Nota: La cancellazione della fattura è possibile solo nel caso in cui non sia già stata registrata in prima nota.

Premere il pulsante Variazione F3, ricercare il cliente nella maschera di ricerca cliente. Premere il pulsante Elimina F10 per rimuovere il cliente dalla base dati.

Fornitori

Permette l'inserimento, modifica e cancellazione dei fornitori.

Inserimento

Premere il pulsante Nuovo F12, inserire i dati del fornitore e premere il pulsante Salva F8. Seguire le indicazioni in caso di dati mancanti o incompleti.

Modifica

Premere il pulsante Variazione F3, ricercare il fornitore nella maschera di ricerca fornitore. Modificare i dati e premere il pulsante Salva F8. Seguire le indicazioni in caso di dati mancanti o incompleti.

Cancellazione

Nota: La cancellazione del fornitore è possibile solo nel caso in cui non sia già stato utilizzato nei moduli di Paperclip.

Premere il pulsante Variazione F3, ricercare il fornitore nella maschera di ricerca fornitore. Premere il pulsante Elimina F10 per rimuovere il fornitore dalla base dati.

Report Clienti/Fornitori

Il report clienti/fornitori permette di elencare e stampare l'elenco dei clienti/fornitori. L'opzione attivi filtra la ricerca solo per i clienti/fornitori attivi.

Ricerca Clienti/Fornitori

La ricerca dei clienti/fornitori è attivabile col pulsante Variazione F3 o con i tasti funzione F6 per il cliente F7 per il fornitore.
